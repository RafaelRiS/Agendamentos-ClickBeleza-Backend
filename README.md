# 💇‍♀️ ClickBeleza - Backend

API responsável pelo gerenciamento de agendamentos do sistema ClickBeleza.

---

## 🚀 Tecnologias utilizadas

- Python
- FastAPI
- SQLite
- SQLAlchemy
- Uvicorn

---

## 📦 Funcionalidades

- Criar agendamento
- Listar agendamentos
- Filtrar por telefone
- Atualizar serviço
- Deletar agendamento
- Verificar conflitos de horário
